# MET API Game

A React-based trivia game that tests your art history knowledge using the Metropolitan Museum of Art Collection API.

---

## About the Game

This application connects to The Metropolitan Museum of Art's open access API to create an educational guessing game. The player is presented with three distinct pieces of art retrieved directly from the museum's collection. The objective is to correctly identify the year in which the artworks were created based on visual cues and historical context.

## Features

- **Dynamic Content:** Fetches real-time data and images from the MET Collection API.
- **Randomized Gameplay:** Every round offers a new set of artwork, ensuring no two games are exactly alike.
- **Responsive Design:** Built with React to work smoothly on desktop and mobile devices.
- **Scoring System:** Tracks correct answers to gauge the player's art history expertise.

## Tech Stack

- **Frontend:** React.js
- **Styling:** CSS / CSS Modules
- **Data Source:** The Metropolitan Museum of Art Collection API
